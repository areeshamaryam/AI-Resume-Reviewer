# AI Resume Reviewer & ATS Analyzer

A Full Stack AI-powered web application that analyzes resumes against job descriptions and provides ATS optimization feedback.

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- OpenAI APInpm install express mongoose cors dotenv bcryptjs jsonwebtoken multer pdf-parse openai
- Tailwind CSS
